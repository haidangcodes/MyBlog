Giới thiệu#

Tôi là sinh viên chuyên ngành Công nghệ thông tin, có định hướng nghiên cứu và phát triển trong lĩnh vực lập trình mạng và ứng dụng Web.
Blog này được xây dựng như một phần của đồ án học phần, đồng thời là nơi tổng hợp và hệ thống hóa kiến thức đã học thông qua quá trình tự nghiên cứu và thực hành.

Kỹ năng và công cụ#

Ngôn ngữ lập trình#

  • Java
  • JavaScript
  • HTML / CSS

Công cụ & môi trường#

  • Git & GitHub
  • Hugo (Static Site Generator)
  • Visual Studio Code

Định hướng phát triển#

  • Phát triển ứng dụng mạng theo mô hình Client–Server
  • Xây dựng Web hiệu năng cao, tối ưu trải nghiệm người dùng

Mục tiêu đồ án#

  • Xây dựng blog cá nhân chia sẻ kiến thức về lập trình mạng
  • Ứng dụng Static Site Generator (Hugo) trong phát triển Web
  • Tối ưu hiệu năng, khả năng mở rộng và tính bảo trì
  • Áp dụng quy trình quản lý mã nguồn với Git

Công nghệ sử dụng#

  • Hugo – Static Site Generator
  • PaperMod – Theme giao diện
  • Markdown – Soạn thảo nội dung
  • GitHub Pages – Triển khai và lưu trữ

Blog được xây dựng nhằm phục vụ mục tiêu học tập, đồng thời tạo nền tảng cho việc phát triển chuyên môn trong lĩnh vực lập trình mạng và Web trong tương lai.
